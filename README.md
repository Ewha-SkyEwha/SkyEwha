# TRENDIE

> 여행 유튜버(인플루언서)를 위한 틱톡, 유튜브의 트렌드를 분석하여 현재 여행 트렌드를 제공하고,
사용자가 업로드 할 영상을 분석하여 (맞춤형) 해시태그와 제목을 제공하는 모바일 어플리케이션

---

## 🔍 프로젝트 개요

**TRENDIE**는
✔️ 최신 여행 트렌드를 시각화한 대시보드
✔️ 사용자 영상의 트렌드 일치도 피드백 보고서
를 제공하는 **여행 유튜버 대상 피드백 제공 서비스**입니다!
